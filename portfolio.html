June 2021 update

 

So far this year I've mostly put my free time into developing the Auto Skeleton script.  However, the head and body geometry in this video was really interesting to get working. I got the idea for the face rig after watching Cloudy with a Chance of Meatballs (see the resemblance?). While trying to model a new character in Maya I noticed that smooth mesh mode felt like animating a face rig. Passing the geometry off to another object to subdivide isn't technically necessary, but it is faster for Maya to run since I don't need to keep the new object's construction history

Skater XL mods C#

 

Skater XL is a new(ish) realistic skateboarding game made by Easy Day Studios. It has a small but loyal fanbase, and even smaller and loyaler modding community. Many nights spent on the SXL modding discord is actually the reason I got into modding in the first place. It's also where I learned more about how Unity Engine works under-the-hood


Escape 2 Quit - mod.io

Escape 2 Quit was my attempt at making a meme mod. When Skater XL was in early access, hitting esc while the game was running was identical to hitting alt+F4, and would quit the application. Modders would make custom pause menus to fix this. Less than a year ago, the developers fixed the problem. So in response I quickly made a simple mod that brought back the old game-ending behavior of the esc key. Some people didn't get the joke... source code here

Santa Mod is a mod for a tutorial series showing how to take a skinned and rigged model and load it into a unity-engine-made game via modding. It also shows how to then assign scripted behavior to your object. In this mod, a "dancing Santa" is spawned into the current map when the user presses F1. Santa will follow the player around the map forever, or until a new map is loaded. The source code is here on my github
